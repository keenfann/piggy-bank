# Piggy Bank

Svensk, self-hostad PWA för att hantera barns sparande (kontant och fond) med roller (förälder/barn), import/export, och full test/CI-pipeline.

## Status

Initial implementation scaffold med:
- Produktkrav och arkitektur
- Datamodell
- API-kontrakt
- CI/CD workflows för test + GHCR
- Synology deployment guide
